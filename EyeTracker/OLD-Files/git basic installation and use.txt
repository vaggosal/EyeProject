Αν όλα έχουνε πάει καλά, θα πρέπει να έχει δημιουργηθεί ο φάκελος "EyeTracker". Ότι ενέργεια μετά (push/pull κτλ) πρέπει να γίνει με το 
τερματικό μας να κοιτάει μέσα στο φάκελο που μόλις δημιουργήθηκε.(του τεστινγ "cd EyeTracker/")


Ποιο μετά....

Έκανα την παραπάνω διαδικασία και μου δημιουργήθηκε ο φάκελος. Πως θα βάλω, Σπύρο, ένα αρχείο μέσα και πως θα το βάλω στο repository;

Δώνουμε ένα $ git config --global user.email "your@email.com"

Φτιάχνω με τον x, y τρόπο ένα αρχείο μέσα στο directory που έχω τοπικά. π.χ. το "git basic installation and use.txt". Το έφτιαξα με το καταπληκτικό notepad.
Στη συνέχεια δίνω..
$ git status
για να δω τι γίνεται. Μου βγάζει ότι υπάρχει ένα αρχείο που όμως δεν παρακολουθείται από το repository. Μου προτείνει gid add και το κάνω
$ git add . ή git add $ git add git\ basic\ installation\ and\ use.txt
Ναι μεν το αρχείο αυτό τώρα "ακουλουθήται" από το git αλλά δεν έχει καταχωρηθεί κάποια αλλαγή. Θα δώσω πάλι $ git status και μου προτείνει να κάνω commit το αρχείο που μόλις προσέθεσα στο repo.
$ git commit -m "Auto einai ena commit minima. PREPEI na bazoume minimata gia na kseroume poy pame.."

Και στη συνέχεια με ένα status πάλι μας ενημερώνει ότι είμαστε μπροστά κατά 1 commit. Ένα commit μπορεί να περιλαμβάνει αλλαγές από πολλά αρχεία. Καλό είναι όμως
κάθε commit είναι από κάτι συγκεντροτικό. Π.χ. $ git commit -m "Allagi twn oriwn twn function"

Με git add βάζουμε ότι αλλαγές. Δεν βάζουμε μόνο καινούργια αρχεία. Π.χ. ένα αρχείο που αλλάξαμε με το status θα μας εμφανίσει ότι έχει αλλαχτεί.(modified)
Το κάνουμε add και στη συνέχεια commit με ένα περιγραφικό μήνυμα.(καλύτερα μεγάλο και περιγραφικό παρά μικρό και μην κατανοητό!)

Τέλος τα commit μας τα προωθούμε στον server/repo με την εντολή
$ git push

Καλό είναι πάντοτε να κάνουμε ένα git pull πριν το push. Να λάβουμε τις ενημερώσεις που άλλοι έχουνε κάνει στο repo και στη συνέχεια να δώσουμε τις δικές μας.
(merge conflict αν έχει αλλαγές το repo όταν κάνουμε pull πάνω στα αρχεία που έχουμε και εμείς τοπικά αλλάξει)

Σε γενικές γραμμές, πριν αρχίσουμε να δουλεύουμε κάνουμε ένα pull και μόλις τελειώσουμε κάνουμε το push.
